# RAON.I 마스터 개발 로드맵 v2 (Integrated Master Roadmap)

**버전**: v2.0 (Integrated)
**기반**: RAONAI SSOT MASTER v9 + External Roadmap v1
**작성일**: 2025-12-05

이 문서는 기존 로드맵의 실행력과 외부 로드맵의 상세한 스펙을 통합하여 만든 **최종 확정형 개발 가이드**입니다.
현재 진행 중인 **내공간(My Space)** 개발 흐름을 유지하면서, **홈(Home)**과 **예약(Reservation)**의 디테일을 대폭 보강했습니다.

## 🚀 상세 로드맵 (Detailed Roadmap)

### Phase 0: 코어 플랫폼 기반 (Foundation) - ✅ 완료
*   **0.1 글로벌 UI 프레임**: TopBar, BottomNav, 390px 레이아웃
*   **0.2 공통 라이브러리**: Shadcn UI, Tailwind, Lucide Icons
*   **0.3 라우팅**: Next.js App Router (`(mobile)`, `admin`)

### Phase 1: 사용자 홈 (User Home) - 🚨 Next Priority
**"라온아이의 첫인상 - 초보자와 숙련자를 위한 맞춤형 대문"**

*   **1.1 분기 엔진 (State Engine)**
    *   [ ] 사용자 상태(신규/기존) 판별 로직
    *   [ ] 홈 히어로 데이터 바인딩
*   **1.2 초보자 홈 (Beginner)**
    *   [ ] **히어로**: 감성 카피 + 전경 이미지
    *   [ ] **정보 칩**: 주소, 길찾기, 배치도, 편의시설 (6~10종)
    *   [ ] **가이드 카드**: 예약부터 퇴실까지 3~5단 안내
    *   [ ] **추천**: 날씨/요리/놀이 큐레이션
*   **1.3 기존 사용자 홈 (Returning)**
    *   [ ] **개인화 히어로**: 내공간 배경 연동
    *   [ ] **예약 패널**: D-Day, 잔여석, "내 텐트로 돌아가기"
    *   [ ] **Slim 공지**: 1줄 롤링 공지
    *   [ ] **커뮤니티 티저**: 최신 인기글 노출

### Phase 2: 내공간 (My Space) - 🔄 Current Focus
**"라온아이의 영혼(Soul) - 1인칭 감성 기록 공간"**

*   **2.1 대시보드 (Dashboard)** ✅
    *   [x] POV 레이아웃 (텐트 뷰)
    *   [x] 액션 위젯 (불멍, 별보기 등)
    *   [x] 요약 그리드 (예약, 포인트, 앨범, 지도)
*   **2.2 나만의 지도 (My Map)** ✅
    *   [x] 지도 연동 및 핀 저장
    *   [x] 장소 상세 시트 (Travel Record Style)
*   **2.3 타임라인 (Timeline)** 🚧
    *   [ ] **감성 리스트 UI**: 단순 텍스트 나열 지양
    *   [ ] **데이터 연동**: 후기, 사진, 예약 이력 통합
*   **2.4 앨범 (Album)** 🚧

*   **2.6 꾸미기 (Decorating)** ⬜

## 💡 개발 원칙 (Development Principles)
1.  **따뜻함 (Warmth)**: 기술보다 감성. "완료되었습니다"보다 "준비해 두었습니다" 사용.
2.  **초보자 배려 (Beginner-first)**: 설명서가 필요 없는 직관적인 UI.
3.  **모바일 퍼스트**: 390px 기준. PC는 확장이 아닌 덤.
4.  **SSOT 준수**: 모든 변경은 `RAONAI SSOT MASTER v9`를 따름.
