## 🧠 AI 적용 전략 (Lightweight Strategy)
*비용 효율성을 위해 단계별로 적용합니다.*
*   **L0 (Rule-based)**: 비용 0원. 규칙 기반 추천, 자동 체크리스트, 제로클릭 재예약.
*   **L1 (Lightweight API)**: 저비용. 텍스트 요약, 태그 추천, 제목 생성 (OpenAI/Gemini API 등).
*   **L2 (Advanced)**: 추후 고려. 대규모 개인화 모델.

## 💡 개발 원칙 (Development Principles)
1.  **따뜻함 (Warmth)**: 기술보다 감성. "완료되었습니다"보다 "준비해 두었습니다" 사용.
2.  **초보자 배려 (Beginner-first)**: 설명서가 필요 없는 직관적인 UI.
3.  **가격 투명성 (Price Decoding)**: 비용의 이유를 친절하게 설명.
4.  **SSOT 준수**: 모든 변경은 `RAONAI SSOT MASTER v9`를 따름.
