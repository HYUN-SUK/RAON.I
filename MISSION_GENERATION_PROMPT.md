# RAON.I 미션 생성 AI 프롬프트 양식

이 문서는 다른 AI(ChatGPT, Claude 등)와 논의하여 RAON.I 앱에 등록할 미션을 생성할 때 사용하는 표준 프롬프트입니다.
아래 내용을 복사하여 AI에게 전달하면, 우리 앱이 이해할 수 있는 **JSON 형식**으로 미션을 만들어줍니다.

---

## 📋 AI 요청 프롬프트 (복사해서 사용하세요)

```text
너는 지금부터 'RAON.I'라는 캠핑 & 힐링 커뮤니티 앱의 "미션 기획자"야.
우리 앱 사용자들이 즐겁게 참여할 수 있는 [주제] 관련 미션을 [개수]개 기획해줘.

각 미션은 아래의 "JSON 표준 데이터 포맷"에 맞춰서 출력해야 해.
다른 설명은 필요 없고, 오직 JSON 배열 코드만 출력해줘.

### 📌 기획 요구사항
1. **타입(mission_type)**: 다음 3가지 중 하나여야 함.
   - "PHOTO": 사진 인증 미션 (가장 많아야 함, 70%)
   - "CHECKIN": 장소/활동 체크인 (20%)
   - "COMMUNITY": 글쓰기/댓글 소통 (10%)
2. **기간**: 일주일 단위로 진행됨.
3. **보상**: 난이도에 따라 XP(100~500), Point(10~100) 설정.
4. **내용**: 30대~40대 캠핑족이 부담 없이 할 수 있는 힐링/감성 테마.
5. **랭킹**: 참여 인증샷은 다른 유저들의 '좋아요'를 통해 랭킹(Top3)이 산정되므로, 사진이 예쁘게 나올만한 주제가 좋음.

### 💾 JSON 표준 데이터 포맷 (Strict)
[
  {
    "title": "미션 제목 (이모지 포함 가능)",
    "description": "미션 상세 설명 (줄바꿈은 \n 사용)",
    "mission_type": "PHOTO", 
    "week_offset": 0,
    "reward_xp": 100,
    "reward_point": 50
  },
  {
    "title": "다음 주 미션 예시...",
    "mission_type": "CHECKIN",
    "week_offset": 1,
    "reward_xp": 150,
    "reward_point": 30
  }
]

**설명**:
- `week_offset`: 0이면 "이번 주(또는 지정한 시작 주)", 1이면 "그 다음 주"에 자동으로 스케줄링됨.
- `mission_type`: 대문자 필수 ("PHOTO", "CHECKIN", "COMMUNITY").
```

---

## 🛠 사용 방법
1. 위 프롬프트의 `[주제]`와 `[개수]`를 원하는 대로 수정하세요. (예: "겨울 장박", "4개")
2. 다른 AI에게 전송합니다.
3. AI가 답변한 **JSON 코드**를 복사합니다.
4. RAON.I 관리자 페이지의 **[미션 일괄 등록]** 메뉴에 붙여넣기 하면 끝!
